jsProSheSim

jsProSheSim is a Javascript library that allows the simulation of a process sheduling system.

The main features are:

  • Multiprocessor support
  • SRT policy
  • Round-robin policy
  • FCFS policy

Live Demo

There is a jsProSheSim sample application available with a nice web interface.

How to use

Include the jQuery and the jsProSheSim.js libraries to your HTML page.

<script type="text/javascript" src="js/lib/jquery-2.1.1.min.js"></script>
<script type="text/javascript" src="src/jsProScheSim.js"></script>

API

var sim = new Simulator();
sim.setProcessors(1);
sim.addQueue('Queue 1', 1, 'FCFS');
sim.addProcess('A', 'Queue 1', 1, 3, [0,1,0,0,1,1,1,0]);
var status = sim.getStatus();
var result = sim.nextStep();
